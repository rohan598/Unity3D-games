### About the game
**1**. It is classic block-breaker arcade game. The more block you break , the more you score.

**2**. It has 6 levels 1 through 6.

**3**. Bricks are of 4 types
  1-hit
  2-hit
  3-hit
  Unbreakable

**4**. There is a pause menu under development
